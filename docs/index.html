1 はじめに

これは、高知工科大学 経済・マネジメント学群で開講されている「統計学1」（担当：矢内勇生）の講義用資料である。

注意

これは授業の補助教材である。

  • 大切なことは授業で説明する。この資料にすべて書かれているわけではない。
  • この資料に書かれていない内容も授業で説明する。よって、この資料に書かれていない事項でも授業で説明した内容はすべて試験範囲である。
  • 受講生全員が 教科書 と KUTLMS で配布する予習・復習用資料 の指定範囲を読んでいるという前提で授業を進め、試験を実施する。教科書と予習・復習用資料（以下、教科書等）の内容をこのウェブ資料で詳しく説明することはないので、必ず教科書等を読むように。
